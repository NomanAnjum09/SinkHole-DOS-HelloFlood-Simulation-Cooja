# SinkHole-DOS-HelloFlood-Simulation-Cooja

#### This project contains simulation of some famous attacks over IOT networks with the help of Cooja simulator. Malicious nodes have been generated with the with name of Flood, Sinkhole and DOS, while the non malicious motes can be added from nodes folder.
#### To observer simulation add sink from node folder and malicious motes from other malicous node folder.
